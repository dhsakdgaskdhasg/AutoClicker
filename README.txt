# AutoClicker

A multifunctional auto-clicker with macro recording (mouse and keyboard), profile support, and bilingual support.

## Features
- Standard auto-clicker with flexible settings
- Macro recording: clicks, mouse movements, keystrokes
- Infinite macro playback
- Hotkey management (F6/F7/F8)
- Multiple profiles with saved settings
- Interface in Russian and English
